In accordance with the JavaBeans specification section 8.3.2:

Boolean properties

In addition, for boolean properties, we allow a getter method to match the pattern: public boolean is();This "isPropertyName" method may be provided instead of a "get" method,or it may be provided in addition to a "get" method. In either case,if the is method is present for a boolean property then we will usethe "is" method to read the property value. An example boolean property might be:

public boolean isMarsupial();
public void setMarsupial(boolean m);
